# Japanese Learning App – Tenjin-Ya

**Tenjin-Ya** is a Japanese language learning application currently being developed as part of my upcoming **Bachelor’s thesis project**.

The goal of this project is to explore and experiment with different learning application features while designing a platform that offers a more engaging and well-rounded approach to studying Japanese. After analyzing existing language learning apps and their functionalities, I combined those ideas with my own concepts to create something unique.

## Overview

The application integrates multiple learning components, including:

- Grammar practice
- Vocabulary learning
- Reading exercises
- Interactive tests and progress tracking
- Cultural insights and introductions to Japanese traditions

By combining structured language learning with cultural context, **Tenjin-Ya** aims to provide a richer and more immersive experience for users who want to study Japanese.

## Project Purpose

This project is part of my **Bachelor's thesis** and focuses on exploring how language learning applications can integrate both **educational structure and cultural immersion** to improve the learning experience.
